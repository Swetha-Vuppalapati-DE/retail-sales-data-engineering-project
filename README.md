# Retail Sales Data Engineering Project



\## 📌 Overview

This project demonstrates an end-to-end data engineering pipeline using Azure services.



\## 🧰 Tech Stack

\- Azure Data Factory (ADF)

\- Azure Data Lake Storage (ADLS)

\- PySpark

\- Azure SQL Database

\- Power BI



\## 🔄 Architecture

CSV → ADF → ADLS (Raw) → PySpark → ADLS (Curated) → Azure SQL → Power BI



\## 📁 Dataset

\- Orders

\- Customers

\- Products
